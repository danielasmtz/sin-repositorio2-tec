CSS

CSS es la mejor forma de separar los contenidos y su presentación y es imprescindible para crear páginas web complejas.

El funcionamiento de las hojas de estilo es el siguiente:
En la página web (archivo .html) se escriben las etiquetas que definen categorías o elementos.
En la hoja de estilo (archivo .css) se escribe cómo queremos que sea el estilo de presentación de las etiquetas (color, tamaño, fuente, bordes, márgenes, posición, etc).
En la página web se escribe qué hoja de estilo queremos utilizar.

Etiquetas en css:

1. color; Color de texto
2. background-color; Color de fondo
3. backgroud:url("/images/back.jpg") repeat-x; Repetir imagen de fondo en forma horizontal
4. background: url("/images/back.jpg") repeat-Y; Repetir imagen de fondo en forma vertical
5. font-weight:bold; Negrita
6. font-style:italic; Cursiva
7. text-decoration: underline; Subrayado
8. text-transform: uppercase; Parafo en mayusculas
9. l.text-transform: capitalize; Letra capital o texto tipo titulo
10. a{text-decoration:none;}: Eliminar subrayado de un enlace
