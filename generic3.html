HTTP

Hypertext Transfer Protocol (HTTP) o Protocolo de Transferencia de Hipertexto.

Esta tecnologia cuenta con las siguientes etiquetas o peticiones:

1. GET: Solicita una representacion del recurso especificado.
2. HEAD: Pide una respuesta identica a la que correscpondia a una peticion GET, pero en la respuesta no se devuelve el cuerpo.
3. POST: Envia los datos para que sean procesador por el recurso identificado.
4. PUT: Sube, carga o realiza un upload de un recurso especificado, un archivo o fichero.
5. DELETE: Borra el recurso especificado.
